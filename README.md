## Автоматизация (GitHub Actions)

В репозитории настроен **GitHub Actions**. При каждом `push` в ветку `main` система автоматически:

1. Находит все новые или изменённые файлы `.tex`
2. Компилирует их в PDF в два прохода (для корректного оглавления)
3. Загружает готовые PDF обратно в репозиторий

> ⚠️ **Примечание:**  
> Сейчас система при каждом push заново компилирует **все** `.tex` файлы, даже если они не изменились. Надо будет настроить обработку только изменённых файлов для ускорения работы
